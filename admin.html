How to Use Admin

Daily Review Workflow

  1. Terminal: node scripts/prep-review.js
  2. Note the 3 parishes. Click their Bulletin URL to open it
  3. Download the latest bulletin PDF; copy its text
  4. Open claude.ai → paste review/today-prompt.txt + the bulletin text
  5. Search the sidebar here, select each parish
  6. Update Mass times in the Schedule section — click Edit on any row
  7. Expand Validation → set status to manually verified, set Last Checked to today
  8. Save Parish after each one
  9. When done → Download JSON, replace the file, commit & push

Schedule Editing

  • Services are grouped by type: Sunday Mass, Weekday Mass, Confession, etc.
  • Click Edit on any row to change its day, time, language, or notes
  • Click + Add next to a group heading to add a new service in that category

Saving Your Work

  • Save Parish writes to memory only — button pulses gold when unsaved edits exist
  • Download JSON exports all parishes; always do this before closing

Status Dot Guide

  • Green — verified or manually verified
  • Amber — needs verification or low confidence
  • Grey — never reviewed
  • Red — excluded from the app

URL Audit

Run node scripts/audit-urls.js to find dead or missing URLs. Fix Website and Bulletin URL here, then Download and commit.
