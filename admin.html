How to Use Admin

Daily Review Workflow

  1. In your terminal: node scripts/prep-review.js
  2. Note the 3 parishes listed — open their source URLs in your browser
  3. Open claude.ai → paste review/today-prompt.txt
  4. Copy any bulletin text or screenshots and include them in the chat
  5. Back here: search the sidebar to find each parish
  6. Switch to the Schedule tab and update times to match the bulletin
  7. Switch to Validation, set status to manually_verified, set Last Checked to today
  8. Click Save Parish after each one
  9. When all three are done → Download JSON
  10. Replace parish_data.json in your repo, then commit and push

What Each Tab Contains

Overview Parish name, status, bulletin URL — start here for URL fixes
Schedule All Mass times, confession, adoration, special events — your main editing view
Locations Physical buildings, addresses, and GPS coordinates
Contact Phone, website, office hours, social media
Validation Review status and date tracking

Saving Your Work

  • Save Parish writes changes to memory only. The button glows gold when you have unsaved edits — don’t switch parishes without saving first.
  • Download JSON exports all parishes to a file you can commit. Always download before closing the tab.

Status Dot Guide (sidebar)

  • Green — verified or manually verified recently
  • Amber — needs verification or low confidence
  • Grey — never reviewed
  • Red — excluded from the app

URL Audit

Run node scripts/audit-urls.js to find parishes with missing or broken website / bulletin URLs. Fix them in the Overview (bulletin URL) and Contact (website) tabs, then Download and commit.
