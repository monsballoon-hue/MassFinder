How to Use Admin

Daily Review Workflow

  1. Terminal: node scripts/prep-review.js
  2. Note the 3 parishes. Click their Bulletin URL in the header bar
  3. Download the latest bulletin PDF; copy its text
  4. Open claude.ai → paste review/today-prompt.txt + the bulletin text
  5. Search the sidebar here, select each parish
  6. Click any day or time in the schedule to edit inline, or click ··· for full edit
  7. Click ✓ Verify in the header to mark as reviewed today
  8. Save Parish after each one
  9. When done → Download JSON, replace the file, commit & push

Keyboard Shortcuts

Save current parish
Download JSON
Alt ↑ / ↓ Previous / Next parish
Alt Q Toggle Quick Scan mode
/ Focus search bar
Esc Close any modal / clear selection

Schedule Editing

  • Services are grouped by type with colour-coded left borders
  • Click a day or time to edit it inline — Enter saves, Esc cancels
  • Click ··· to open the full editor for language, notes, seasonal flags, etc.
  • Click + Add next to a group heading to add a new service (inherits language from group)

Saving Your Work

  • Save Parish writes to memory only — button pulses gold when unsaved edits exist
  • Download JSON exports all parishes; use at end of session
  • Export Changes exports only parishes you edited this session (patch file)
  • Import merges a previously exported changes file into current data
  • Closing the tab with unsaved changes will trigger a browser warning

Sidebar & Bulk Actions

  • Click the sort button to cycle: Priority (stalest first), A→Z, or Status
  • Shift+click parish rows to multi-select, then Verify All to mark them all as verified today
  • Progress bars show verified/total services for parishes you’ve visited this session
  • Quick Scan (header button or Alt Q) hides all cards except Schedule for speed

Status Dot Guide

  • Green — verified or manually verified
  • Amber — needs verification or low confidence
  • Grey — never reviewed
  • Red — excluded from the app
